# 🚀 SkillPath – React Website Enhancement

> **Assignment 1** – Lenovo LEAP Web Development Internship

## 📖 Project Overview

SkillPath is a responsive multi-page React application developed as part of my **Lenovo LEAP Web Development Internship**. This project demonstrates the fundamentals of modern React development, including reusable components, client-side routing, state management, controlled forms, and responsive UI design.

The application provides a clean and interactive learning platform interface with multiple pages and seamless navigation.

---

## ✨ Features

* 🏠 Responsive Landing Page
* ℹ️ About Page
* 📩 Contact Page with Form Validation
* 🚫 Custom 404 Not Found Page
* 🧩 Reusable React Components
* 🧭 React Router Navigation
* 📱 Mobile-Friendly Responsive Design
* ✅ Controlled Forms using `useState`
* ⚡ Event Handling with `e.preventDefault()`
* 🎯 Dynamic Rendering using Props
